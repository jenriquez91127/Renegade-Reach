# Bakersfield College — Typography

## Institutional Typeface

**Proxima Nova** — the official BC typeface, confirmed in the Brand Usage Guidelines.

- Classification: Geometric sans-serif
- Designer: Mark Simonson
- Character: Clean, rounded, highly legible — modern without feeling cold
- Weights available: Thin, Light, Regular, Semibold, Bold, Extrabold, Black (+ italics)
- License: Must be installed via BC's official distribution ("How to Install Proxima Nova Font" referenced in brand guide)

## Usage Guidelines
- Use Proxima Nova for all official BC communications and web pages
- The BC website uses sans-serif throughout — no serif type in the primary identity
- Heading hierarchy uses varying weights of Proxima Nova (Bold/Extrabold for display, Regular/Light for body)
- Proxima Nova is NOT a free font — for web use, source via Adobe Fonts (Typekit) or a licensed CDN

## Web Fallback Stack
```css
font-family: 'Proxima Nova', 'proxima-nova', 'Helvetica Neue', Arial, sans-serif;
```

## Notes
- Do not substitute Inter, Roboto, or other geometric sans-serifs as primary type
- Proxima Nova has strong visual kinship with Futura and Gill Sans but with better screen rendering
